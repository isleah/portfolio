PlayList Page

Playlist Page

This was a personal project done in HTML, CSS, and JavaScript. It was an edited version of Brad Traversy’s Sound Board project from an education series he did to teach the basics of Web Development. I made the project while learning JavaScript and it was a fun way to learn and call functions. The fonts, backgrounds, and design were all done in pure CSS and I had so much fun with the design I made two versions. The original source code was posted to Tumblr for personal use.



Form Project

A Form Design project done in HTML & CSS as an assignment for Nucamp. At Nucamp code school, they taught us the basics of front-end design, especially elements we would be encountering often, like forms. The project was a showcase of all types of form inputs such as checkboxes, radio types, and how to build a proper form with the right attributes. The original design of the form was simple, so I used CSS to make the purple design along with the bolder fonts and larger input fields.

Bootstrap Console

This was a personal project I had made to practice using Bootstrap, a CSS framework. Bootstrap is a CSS framework that uses prewritten code to build sites faster for developers. This was a great introduction into using package managers and my first time learning a framework. The project uses the basic building blocks of Bootstrap, such as rows and columns, grids, cards, and list groups. The page is a progress tracker for the game Pokémon Legends: Arceus. The pokemon at the top were my current team, and the list card was my progress throughout the game.

Blog Website

This Blog Website project was done for the class CMST 386, Principles of Web Design and Technology II. It is a fictional fan site I called DemonSlayerCorps.com. It was a multipage fan blog showcasing the characters and story arcs of the Demon Slayer series. It included pages such as a list of the main protagonists, a merchandise page, and a movie page with playable trailers. I used a few pieces of fan art for visuals as well as official art for the characters and movies. The project was written in HTML, CSS, and Bootstrap 5 and is designed for desktop and mobile viewports.

Landing Page

This personal project was a landing page design I created with photoshop, HTML, and CSS. My design uses the graphic to the left as an anchor for the color and style of the landing page. The page was meant to be a design for a fashion page or blog and showcases Dua Lipa as the model in the photo. Together with the simple links and icons, the graphic was made in photoshop using a variety of blends and layer styles. I experimented with individual colors, brightness and contrast, and other blending methods to create an eye-catching visual for the landing page.

Mobile App Prototype

This mobile app project was done for the class CMST 308. We were required to go through the design process of building a mobile application, complete with wireframes, storyboarding, and persona creation. My app was called “Coffee Filter” and functions as a search and filter engine for coffee shops in a designated area. The application was a fun introduction into user interaction design and the extensive research that happens during the design process. Some pages included a search the map function, a write a review page, and a way to add “tags” to shops to provide a description of the shop’s menus and amenities.

Menu Re-Design

This menu redesign was one of my first paid projects. A former friend I worked with reached out to collaborate with me to design a menu for a fast casual restaurant called Pizzeria Kitchen. I used the digital assets he had provided to create a new design that used vibrant pictures, colors and legible typography. I tried to make the menu seem not as text heavy by using a mix of bold texts and slim fonts. This was also my first time using Canva in a collaborative project, and I had to learn Canva’s interface quickly to submit the project within the deadline.

Illustrator Design

This project I made during my time in community college in an Adobe Illustrator course. It is a design of Brook from the manga One Piece. I learned to use the pen tool and how to switch through shortcuts quickly in this course, and I think my Illustrator skills have significantly improved since then. Since then, I have learned how to create SVGs, vectors, use image tracing, and use patterns, which I showcase in other projects during my time at UMGC.

Movie Poster

This movie poster design was made for CMST 325. It is one of my favorite projects because it uses images from one of my favorite games, Miles Morales for PlayStation 5. The project was done in photoshop and uses screenshots I took from my own console. The color scheme of black, red, and blue pay homage to Spider-Man’s colors, and white text to contrast. I used gradients for texture on the project, and experimented with the blur and dodge tools to create a dramatic tone for the project.
