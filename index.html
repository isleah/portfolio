Mobile App Prototype

A Mobile Application mock up for a class project. We were required to go through the design process of building a mobile application, complete with wireframes, storyboarding, and persona creation. My app was called “Coffee Filter” and functions as a search and filter engine for coffee shops in a designated area. The application was a fun introduction into user interaction design and the extensive research that happens during the design process. Some pages included a search the map function, a write a review page, and a way to add “tags” to shops to provide a description of the shop’s menus and amenities.

Menu Re-Design

This menu redesign was one of my first paid projects. A friend I worked with reached out to collaborate with me to design a menu for a fast casual restaurant called Pizzeria Kitchen. I used the digital assets he had provided to create a new design that used vibrant pictures, colors and legible typography. I tried to make the menu seem not as text heavy by using a mix of bold texts and slim fonts. This was also my first time using Canva in a collaborative project, and I had to learn Canva’s interface quickly to submit the project within the deadline.

Movie Poster

This movie poster design was made for a class project. The project was done in photoshop and uses screenshots I took from my own console. The color scheme of black, red, and blue pay homage to Spider-Man’s colors, and white text to contrast. I used gradients for texture on the project, and experimented with the blur and dodge tools to create a dramatic tone for the project.
